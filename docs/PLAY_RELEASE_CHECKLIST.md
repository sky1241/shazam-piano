# Play Release Checklist

- [ ] Bump versionName/versionCode and confirm release notes.
- [ ] Verify release signing (keystore, upload key, Play App Signing).
- [ ] Internal testing track configured and testers added.
- [ ] Data safety form completed (microphone, ads, analytics, purchases).
- [ ] Privacy policy URL ready and updated in-app (Settings > Confidentialite & donnees) when available.
- [ ] RECORD_AUDIO permission rationale and denied flow verified.
- [ ] AdMob placements reviewed (banner + one-time interstitial only; test IDs in debug/profile, real IDs in release).
- [ ] IAP disclosure and restore flow verified.
- [ ] Store listing assets ready (icon, screenshots, feature graphic).
- [ ] google-services.json remains untracked per repo policy.
